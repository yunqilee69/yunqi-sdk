## 介绍
yunqi-sdk是自己编写的一些sdk，可以在项目中引用。

想要做出下面的通用sdk，供自己使用和网友学习

### yunqi-sdk-oss
面向对象存储的sdk，集成阿里云、腾讯云、MinIO三种（后续可继续添加厂商)

### yunqi-sdk-sms
封装短信厂商，集成阿里云、腾讯云两种（后续可继续添加厂商）。

### yunqi-sdk-email
将邮箱的一些操作进行封装，并添加FreeMarker（其他的模板框架后续也可以支持）的方式进行格式化邮件内容。

### yunqi-boot-starter-XXX
将上面的sdk提供spring boot版本，在spring boot项目直接引入依赖和配置文件即可使用
